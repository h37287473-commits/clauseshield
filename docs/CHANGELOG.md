# 版本记录 (Changelog)

## [0.1.0] - 2026-04-23 — MVP 版本

### 新增

#### 核心功能
- **6 Agent 并行扫描引擎**：IP归属、付款节点、无限责任、维护范围、保密期限、竞业限制
- **合同条款自动切分**：基于语义断句的智能分句器
- **PDF 文本提取**：基于 pdfplumber 的文本层提取
- **风险分级报告**：高危 / 警告 / 提示 / 安全 四级可视化

#### 数据与模型
- **Few-Shot 示例库**：中英文双语示例库，提升常见陷阱识别率
- **6 个专业 Agent**：每个 Agent 独立提示词 + 示例注入

#### 合规与支付
- **合规包装模块**：自动生成法律声明、AI 生成内容提示、标准建议模板
- **兑换码支付系统**：纯本地 JSON 数据库，零外部支付 API 依赖
- **免费试用机制**：新用户享一次完整扫描体验

#### 防刷与安全
- **IP 频率限制**：单 IP 每日最多 5 次扫描
- **文件大小限制**：最大 10 MB
- **内容长度限制**：最大 50,000 字符
- **文件类型白名单**：.txt / .pdf / .doc / .docx
- **内容安全检查**：拒绝空文件、乱码文件
- **敏感词过滤**：基础暴力/色情/政治敏感词列表

#### 前端与体验
- **Streamlit 应用界面**：侧边栏上传 + 主内容区报告
- **分级报告展示**：完整版 vs 受限摘要版
- **报告导出**：文本报告 + 合规完整报告（带 UTF-8 BOM）
- **中英文切换**：支持中文与英文扫描模式

#### 工程与文档
- **GitHub 仓库初始化**：README、LICENSE (MIT)、.gitignore
- **Issue 模板**：Bug 报告 + 功能请求模板
- **FAQ 文档**：覆盖免费策略、格式支持、准确率、数据安全
- **微信接入指南**：公众号自动回复配置说明

### 已知限制
- 仅支持文本层 PDF，不支持扫描件/图片 PDF
- 内存存储限流数据，重启后重置
- 兑换码数据库为本地 JSON，无分布式支持
- 敏感词列表为基础版本，未接入实时更新
