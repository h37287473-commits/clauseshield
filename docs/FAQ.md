# 常见问题 (FAQ)

## 1. ClauseShield 是免费的吗？

**新用户可享受一次免费完整扫描体验。** 之后需要输入兑换码解锁更多扫描次数。

兑换码可通过以下方式获取：
- 关注微信公众号获取限时兑换码
- 参与社区活动或推荐计划

---

## 2. 支持什么格式的合同文件？

目前支持以下格式：

| 格式 | 支持状态 |
|------|---------|
| `.txt` | ✅ 完全支持 |
| `.pdf` | ✅ 完全支持（文本PDF） |
| `.doc` | ✅ 支持 |
| `.docx` | ✅ 支持 |

**不支持**：扫描件/图片PDF、加密PDF、手写合同照片。

**文件限制**：
- 最大文件大小：10 MB
- 最大文本长度：50,000 字符（超长合同建议分段上传）

---

## 3. 扫描准确率多少？

ClauseShield 的识别基于以下机制：

- **6 个专业 Agent**：每个 Agent 聚焦单一风险维度
- **Few-Shot 示例增强**：内置真实合同陷阱案例库
- **置信度评分**：每条风险附带 0-100% 的置信度分数

**准确率参考**：
- 高危风险（置信度 ≥80%）：对常见陷阱的识别率约 **85-95%**
- 警告风险（置信度 60-79%）：对模糊条款的提示率约 **70-85%**
- 提示风险（置信度 <60%）：存在误判可能，建议人工复核

> ⚠️ **重要**：AI 识别存在固有局限，所有结果都应经过人工复核。ClauseShield 不做法律建议。

---

## 4. 我的合同数据安全吗？

**ClauseShield 不保存用户上传的合同原文。**

- 合同文本仅在内存中临时处理，扫描完成后立即释放
- 扫描结果仅在用户浏览器会话中保留，刷新页面即清除
- 不对任何第三方传输或存储合同全文
- 仅向 AI API 发送需要分析的条款片段（不包含完整上下文）

**注意**：
- 请勿上传包含真实商业机密或敏感个人信息的合同
- 建议使用脱敏后的测试版本进行扫描
- 服务器日志可能记录文件元数据（文件名、大小、扫描时间），但不记录内容

---

## 5. 扫描结果为"无风险"，合同就一定安全吗？

**不一定。**

"未发现风险"仅表示当前扫描维度未命中已知风险模式，不代表合同绝对安全：

- 可能存在未纳入当前风险库的新型陷阱
- 行业特定条款（如医疗、金融、跨境）可能超出识别范围
- AI 对复杂法律语境的理解存在边界

**结论**：无论扫描结果如何，签署合同前咨询专业律师始终是必要步骤。

## 6. 兑换码用完了怎么办？

每个兑换码有使用次数上限（通常为 1-5 次）。用完后可：

1. 关注微信公众号获取新兑换码
2. 等待每日重置（如适用）
3. 通过社区推荐计划赚取额外次数

---

## 7. 支持英文合同吗？

**支持。** 在侧边栏选择 "English" 即可切换英文扫描模式。英文模式会加载英文 Few-Shot 示例库，输出英文报告。
